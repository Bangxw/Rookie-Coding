# CSS3 Filter

|||取值范围|
|:-|:-|:-|
| grayscale | 灰度 | 0-1 / 0%-100% |
| sepia | 褐色 | 0-1 / 0%-100%  |
| saturate | 饱和度 | |
| hue-rotate | 色相旋转 | 值为angle(0deg-360deg) |
| invert | 反色 | 0-1 / 0%-100% |
| opacity | 透明度 | 0-1 / 0%-100% |
| brightness | 亮度 | 值为0-1之间的小数 |
| contrast | 对比度 | 值为num |
| blur | 高斯模糊 | 值为length |
| drop-shadow | 阴影 |  |

## saturate()

> 转换图像饱和度。 值为0%则是完全不饱和，值为100%则图像无变化。超过100%（>100%）也是允许的，则有更高的饱和度
